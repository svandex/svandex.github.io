TGA2015大奖

发表于 2015-12-04 更新于 2019-01-05

曾经的大奖也开始逐渐默认，让我们来一起认识一下Video Game Award的继任者The Game Award。

TGA来源

The Game Award最早源自于Video Game Award，是美国SpikeTV电视台从2003年开始举办的一年一度电视游戏大奖，Geoff Keighly为制作人，是除了PlayStation Award和Game Developer Conference Award(GDC Award)之外最有名的电视游戏奖项。2014年SpikeTV电视台没有继续支持该奖项，所以由Geoff Keighly自发地和众多游戏媒体一起举办了寒酸的一届电视大奖，将该奖项改为The Game Award，所以今年是TGA大奖的第二年了。请不要和腾讯的那玩意混一起。这是今年的VGA宣传片。不过确实没有了SpikeTV的支持之后，少了很多大作的World Primere，去年和今年的大奖其实看起来都挺寒酸的。

各种大奖

本次大奖将会公布年度游戏(Game of the Year)，年度最佳开发者(Developer of the Year)，最佳游戏玩家(Trending Gamer)等诸多奖项。我个人比较关注的就是前面的这几个奖项，年度游戏的视频如链接所示。今年确实是一个大作年。狩魔猎人3(The Witcher 3: Wild Hunt)，合金装备5(Metal Gear Solid: Phantom Pain)，辐射4(Fallout 4)，血源(Bloodborne)和马里奥制造(Super Mario Maker)，都是可玩性极高而且制作精良的大作。

  • 狩魔猎人3有着宏大的游戏背景，剑与魔法的世界观，前两代积攒下来的人气和三部曲终章给人遗留下来的悬念，都有着巨大的吸引力。这个系列在我心中也一直是神作，而且我一直很有想法能够重制一下第一代的作品，毕竟第一代不仅年代悠久，2007年的老游戏了，而且战斗方式确实有点奇葩。我希望能够通过重制的方式，把这款游戏的魅力让更多的新玩家体验到。
  • 合金装备已经是一个很悠久历史的品牌了，1998年推出Metal Gear Solid开始凭借着小岛秀夫独特的游戏导演技巧，高超的故事叙述手法，创新的潜入式玩法，风靡全世界，我觉得算是今年最能挑战狩魔猎人3成为年度游戏的游戏了。
  • 辐射4最早也是起源于1998年的Fallout，类似于暗黑破坏神3(Diablo 3)的45度视角，独特的核战后启示录世界观也非常具有吸引力，不过在我心中一直觉得辐射系列是耐玩但不一定好玩的游戏，通关了Fallout 3和Fallout New Vegas，两个游戏加起来玩了200多个小时，没有太多令人印象深刻的故事，有的只是不断地探索的感觉。
  • 血源也是我很喜欢玩的游戏，自从通关Dark Souls之后，心中就一直惦记着什么时候可以上手血源，上个月末已经出来了年度版，这个圣诞就可以打折入手了，嘿嘿。
  • 马里奥作为Nintendo的招牌游戏，游戏性不用说，看看熊猫Tv上多么火爆地直播这款游戏就知道了。同时也很期待明年的NX新主机。

大奖晚会中的小插曲

小岛秀夫这次没有来TGA，原因是Konami通过法律手段限制了小岛秀夫参加本次活动，不得不说是种遗憾，当然我估计很多老美都会出视频大骂Konami的，Konami近几年真的是每况愈下，不仅转型手游，而且曾经的几大王牌游戏都开始力不从心，还敢裁员小岛秀夫，我也觉得是不是高层脑子进水了。Geoff在VGA现场说明了小岛没有来原因。另一个我觉得比较有意思的是美国著名脱口秀，哈弗大学高才生，从大学就喜欢开始恶搞的Conan O’brien通过视频的方式宣布了年度游戏工作室的候选，这个逼我一直觉得挺逗的。还有前IGN雇员和当家主持Craig Miller被评选为最佳游戏主播，国内这么翻译的，不过老外叫做Trending Gamer，就是潮流玩家。其实一直感觉Craig是那种带点装逼性质的逗逼，算是比较有个性，但是他新的Kindafunny Games的播客我基本上没有看完的。

喷点

有句话说的好。日本鬼子，韩国棒子，中国喷子。。。。

我有必要喷一点东西。游戏预告片毫无亮点，Quantum Break的预告片和翔一样，也就30秒时间跟电视广告一样看不出啥东西。Ubi的Farcry Primal的首次实际演示，感觉玩法没变换了个环境而已，不好玩依旧是不好玩。其他预告片我完全没有印象了，本来说有Mortal Kombat X的预告片的好像还没放，网上还挺多人喷这事的，还是等明天凌晨两点的PlayStation Experience上看十款新作好了。
