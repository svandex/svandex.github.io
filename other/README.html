Blog Posts of My Personal Blog Site

md files for hexo-based personal blog

How To Use

  • Install node.js and npm, then npm install hexo-cli -g to install hexo
  • Use hexo init to initialize a directory, which should be an empty directory, use npm install to install hexo-related npm dependent packages
  • git clone this repository into source directory to <path/to/personal>, git clone a theme repository into themes directory.
  • Use hexo g and hexo s to construct the site locally, use http://localhost:4000 to see the site.
  • Use hexo d to deploy the site to github repository. Before deployment, you need to setup _config.md file in hexo initialized directory.
  • In _config.yml file, set source key value to <path/to/personal>, and change some values in _config.yml in hexo directory and in personalblog directory
  • copy hexo.yml to root directory and rename it to_config.yml
  • copy custom.styl to <rootpath>/themes/next/source/css/_custom to customize header font size and indentation.
