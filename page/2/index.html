第七世代十大游戏

发表于 2015-12-15

电视游戏有世代(generation)的说法，第七世代就是指PS3，XBOX360和Wii了。盘点一下本人心中的十大经典游戏，从第十名往第一名数起。

Medal of Honor

还记得大二(2010)时候参加荣誉勋章的多人Beta，激动万分甚至可以彻夜通宵。发售后仍然挡不住我玩了一百个小时的多人对战。当时看了这个多人模式宣传片，真是激动万分。MOH的单人模式是由DICE亲自制作的，讲述的是阿富汗战争中的一个小队执行一系列任务的故事，讲述了美国大兵在阿富汗付出的代价是很沉重的，这个小队最后只剩下了一个队员，每个其他队员都为了美国的“国家安全”做出了不朽贡献。多人模式是Danger Close工作室做的，现在又改名为DICE LA，也是Battlefield 4的后续DLC和“打补丁”工作室。我最喜欢的是里面的枪械音效特别的棒，杀人得分的时候那个亮分的动画特别带感，还有击中时候一个小叉叉也很带感。最有意思的就是狙击手的Mortar了，简直是BUG的存在，运气好可以一次杀五个以上。最后欣赏一下发售宣传片。

聊聊游戏界的那些事

发表于 2015-12-14

明天就是PlayStation Experience了，真想飞到美帝去参加盛会。每年都有大量的电子游戏会展会在全世界各地展出，今天我就想趁这个机会聊聊游戏界的电子游戏展会，聊聊电子游戏界的工作室大牛们。

三大会展

说到三大会展，就是E3、TGS、Gamescom了。早年也有人说有CJ，我一般都对这个看法持呵呵的观点。当然Gamescom是后起之秀了，算是在2011年DOTA2在Gamescom上首秀之后，越来越多的游戏厂商选择了在每年的8月份在GamesCom上宣传自己的游戏了。

TGA2015大奖

发表于 2015-12-04

曾经的大奖也开始逐渐默认，让我们来一起认识一下Video Game Award的继任者The Game Award。

TGA来源

The Game Award最早源自于Video Game Award，是美国SpikeTV电视台从2003年开始举办的一年一度电视游戏大奖，Geoff Keighly为制作人，是除了PlayStation Award和Game Developer Conference Award(GDC Award)之外最有名的电视游戏奖项。2014年SpikeTV电视台没有继续支持该奖项，所以由Geoff Keighly自发地和众多游戏媒体一起举办了寒酸的一届电视大奖，将该奖项改为The Game Award，所以今年是TGA大奖的第二年了。请不要和腾讯的那玩意混一起。这是今年的VGA宣传片。不过确实没有了SpikeTV的支持之后，少了很多大作的World Primere，去年和今年的大奖其实看起来都挺寒酸的。

新开的博客

发表于 2015-11-20

之前用过很多其他站点的博客，但是都觉得是不是非常好用，从本科开始一直很想要开一个自己的博客，无奈当时忙于其他学习也无心花大把的时间配置远程服务器和博客的装饰。转眼现在都快研究生毕业了，才开始制作属于自己的博客。

写博客的主要目的还是写一些和兴趣相关的东西，比如游戏，科技，汽车之类的。我不太想写一些走心的东西，毕竟把自己暴露地太多会让人反感，自己也不想赤裸裸地让别人看个光。博客主要还是会写一些游戏测评相关，电子产品测评相关，还有关于编程方面的。
